  • Getting Started

    Thank you for being a part of our first Voice Drive Audition! Recording should take approximately 30 minutes to read 100 sentences, designed to capture all of the sounds and sound combinations in English.

    All you'll need to record is internet access, a high quality microphone, and a computer with the software Audacity installed (free download for Mac OS X and Windows). For instructions on using Audacity, please refer to this online document. Please remember to speak in a neutral voice, with as little inflection as possible. Speaking too expressively actually results in a poorer sounding voice.

    Next →

  • Setting up Audacity

    After first opening Audacity, you will see a menu in the bottom left corner of the interface for the Project Rate (Hz). The higher the number, the better. 48,000 Hz or better is ideal. Click the red circle icon in the upper left hand corner and navigate back to this window to start recording. We'll wait!

    On the next slide, you'll see a sentence. Read the sentence in a clear, neutral voice before moving onto the next slide. For the first ten sentences, you have the option of clicking the Play Example button to hear an example of the sentence being read. To return to a previous sentence or move onto the next sentence, use the Back and Next arrows. Do not worry about making mistakes when you record - there is no need to restart or edit the recording, so long as it is done in a quiet environment with a good microphone.

  • Submitting your entry

    Congratulations, and thank you. You've taken your first big step toward banking your voice. Go to File > Export on Audacity to save your recording, making sure that the file's format type is set to WAV. Name your file in the following way:

    [initials]_[age]_[gender]_[state]_HVBaudition_WVD.wav

    The next step is to submit your audition via Dropbox, a free online storage and file sharing tool. You'll first need to create an account (if you haven't already). Once youre logged in, create a folder with the same name as your .wav file, and then upload the recording. You can just drag and drop the files into the folder. To share the link with us, right-click on the folder or file name to open the menu, and then click the Share link button. A prompt to send the file will appear. Enter humanvoicebank@gmail.com into the "Add names or emails" field, and click Send.
